💳 Credit Card Analysis Dashboard
📌 Project Overview

The Credit Card Analysis Dashboard is a data analytics project designed to analyze credit card transaction data and customer spending behavior. The dashboard provides insights into customer demographics, spending patterns, transaction trends, and revenue generation.

Using interactive visualizations, this dashboard helps financial institutions understand customer behavior and make better business decisions.

🎯 Project Objectives

Analyze customer spending patterns

Identify high-value customers

Monitor transaction trends

Understand customer demographics

Track revenue generated through credit card usage

🛠️ Tools & Technologies Used

Power BI – Dashboard creation and data visualization

SQL – Data querying and transformation

Microsoft Excel / CSV – Data storage and preprocessing

DAX – Calculations and data modeling in Power BI
